[INDEX_04] Specifications
Main Board ARDUINO UNO R3, 8 bit integer caps.
Clock Speed 16 MHz.
Flash Memory 32 KB (0.5 KB reserved for bootloader)
SRAM 2 KB, PROGMEM optimised for game state
Display 20 x 10 LED STRIP LIGHTS · WS2812B model
Input 4 push buttons + 1 analog joystick
Audio Piezo buzzer · PWM pin 3 · 8-bit driver
Power Nothing and · LDO regulated 5V rail
ARDUINO Dimensions IDK on this one really
Programming C++ · avr-gcc · Arduino IDE compatible
Communication Literally some wire and prayers
Enclosure Industrial · ergonomic handheld form factor
